# Sistema MOPED - Lista de Tareas

## Fase 1: Estructura Base ✅
- Instalar componentes shadcn necesarios
- Crear estructura de carpetas
- Configurar base de datos (JSON)
- Crear tipos TypeScript

## Fase 2: Autenticación ✅
- Sistema de login unificado
- Gestión de roles (ADMIN, SUPER ESPECIALISTA, ESPECIALISTA)
- Protección de rutas

## Fase 3: Sistema MOPED ✅
- Ficha de Gestión Pedagógica (23 ítems)
- Ficha de Planificación Curricular (13 ítems)
- Ficha de Desempeño Docente (5 ítems)
- Control de momentos (1, 2, 3)
- Asignación de IEs a especialistas
- Sistema de 16 redes educativas (REI)

## Fase 4: Gráficos y Estadísticas
- Gráficos por especialista
- Gráficos por nivel educativo
- Dashboard SUPER ESPECIALISTA con gráficos
- Dashboard ADMIN con gráficos

## Fase 5: Admin - Gestión
- Gestión de usuarios (crear, editar, eliminar)
- Configuración de momentos
- Gestión de redes educativas

## Fase 6: Repositorio
- Gestión de documentos PDF
- Enlaces de YouTube
- Categorías dinámicas
- Upload de archivos

## Fase 7: CMS y Página Principal
- CMS para noticias y acontecimientos
- Página principal pública
- Gestión de contenido dinámico
