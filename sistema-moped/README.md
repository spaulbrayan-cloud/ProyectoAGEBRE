# Sistema MOPED

Sistema de Monitoreo Pedagógico y Gestión Educativa (AGEBRE).

Este repositorio contiene la base del sistema construida con:

- Next.js 15
- TypeScript
- Tailwind CSS
- Componentes UI basados en shadcn/ui
- Gestión de estados y contexto para autenticación y roles

Para más detalle funcional, revisa `.same/README.md`.
